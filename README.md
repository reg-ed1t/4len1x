### About
Simple python sockets message system with nicknames

### Requirements
Install the required Python Modules in your machine.
```
pip3 install -r requirements.txt
```
### Installing

##### Server
```
python3 server.py
```
##### Client
```
python3 client.py
```
### Logging

All errors are logged and stored in ` log.txt `
